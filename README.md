## 🌟アプリについて

こちらは、特別に機能がイケてる訳ではなく、文字通り「タスクを達成すると***イケ***メンが褒め***て***くれ***るTODO***」です。

**あ、石投げないで……。**

**あなたの承認欲求を満たすこと**を命題としたTODOアプリです。⇒　元は友人の承認欲求（詳しくは記事にて記載予定です。）

タスクを達成するごとに、承認欲求が満たされれば、継続する力になるっ！！（力説）……はず……。

コーディングより、**イケメンの台詞の作成**に時間がかかったのも良い思い出……。
